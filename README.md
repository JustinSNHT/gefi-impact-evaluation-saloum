# Évaluation d'impact du projet GEFI — Delta du Saloum

Évaluation d'impact (traitement/contrôle) du projet GEFI sur l'autonomisation
des femmes, les revenus halieutiques, la sécurité alimentaire et les
comportements environnementaux dans cinq îles du Delta du Saloum, Sénégal.

## 🚧 Statut
Projet en construction — cadrage terminé, voir [`docs/01_framing.md`](docs/01_framing.md).

## Méthode
Comparaison traitement/contrôle en coupe, estimateur principal par entropy
balancing. Trois restitutions : analyse Python, réplication R, dashboard Power BI.

## Données
Enquête ménage conduite par le CRDES, Dakar. Données brutes non versionnées
(PII). Voir [`data/README.md`](data/README.md).
